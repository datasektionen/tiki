# What is Tiki?

Tiki is an event and ticket management platform for organizations that run ticketed events. It gives you a single place to create events, sell tickets, track orders, and check in attendees.

## Who manages Tiki

In order to be able to manage events, you will need to be part of a team. Ask a team administrator to
give you access. If you want to create a new team, ask a tiki admin to create one and add you to it.

There are two main roles when part of a team:

- **Admins** can manage everything: create and edit any event, manage team members.
- **Event managers** They can manage tickets, orders, and attendees for team events.

## How a typical event works

1. An team admin admin or manager creates an event and sets up one or more ticket types
2. Optionally: A release is created to control when tickets go on sale and in what quantity
3. Attendees purchase tickets through the public checkout
4. At the event date, the team uses the check-in view to scan QR codes and admit attendees
